Corridor generation
While working on ITHOM, it became necessary to develop a method of connecting disjoint rooms with corridors in a gridded world. I experimented with several algorithms and detailed what I learned.
Homebrew recommendations
Ever since I made the switch from Windows to MacOS, I have been using Homebrew as my package manager since day one. It is a fantastic collection of resources which hugely magnify the capabilities of the terminal. I list some of the formulae I have found that are either useful or interesting in some way.

2019

Sample game design document
A game design document is an essential component of an successful video game pitch and a great method of scoping and tracking the process of development. I provide a sample outline of a proper design doc in LaTeX format and give a short description of each section. Feel free to reproduce and distribute as needed.
